About this Site

This data provides a window into how people are interacting with the municipal government online. The data comes from a Google Analytics account for the Municipality of Anchorage. This program helps government agencies understand how people find, access, and use government services online. This program does not track individuals, and anonymizes the IP addresses of visitors.

Please contact the webmaster with comments, questions, or suggestions for improving this application.

This open source project is in the public domain, which means that this website and its data are free for you to use without restriction. You can find the code for this website and the code behind the data collection on GitHub.
